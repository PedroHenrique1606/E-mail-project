## Projeto de envios de E-mails automáticos
Esse projeto tem por finalidade automatizar o envio de e-mails usando o Python 3. Para conseguir obter 100% do funcionamento do código é necessário que você possua um E-mail google e faça algumas breves configurações iniciais.

## Algumas especificações
Nas configurações do seu Gmail, vá para a área de segurança e ao achar a opção `SIGNING IN TO GOOGLE`, desative as duas etapas de verificação, sei que parece arriscado fazer isso, porém é necessário para que o programa faça login na sua conta para poder assim ele enviar o(s) e-mail(s) de fora automática. Ah, também ative a função `acesso menos seguro ao aplicativo`, para que o Python possa entrar no seu E-mail e realizar o login.
